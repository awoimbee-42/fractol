# Fractol
A multithreaded fractal viewer written in C and using 42s graphical library, the MinilibX

#### How to build:
The build process is the same on MacOS and Linux.

    git clone https://framagit.org/awoimbee_42/fractol.git;
    cd fractol;
    make;
    ./fractol mandelbrot
